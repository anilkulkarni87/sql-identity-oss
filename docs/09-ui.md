# Web UI

The web UI provides:
- Setup Wizard for initial configuration
- Dashboard for run metrics
- Explorer for cluster investigation

Run API only (no standalone UI bundle):
```bash
idr serve
```
Open API docs at http://localhost:8000/docs.

Run full API + UI stack:
```bash
docker compose up --build
```
Open UI at http://localhost:3000.

## Setup Wizard
Steps:
1. Connect: choose platform and provide credentials
2. Discover: list tables and columns
3. Map: map identifiers and attributes
4. Rules: configure deterministic + fuzzy rules
5. Survivorship: configure golden record strategies
6. Review: save config and run

## Explorer
- Search by email/phone/ID
- View cluster graph and edges

## Dashboard
- Run summary and metrics
- Confidence scores
- Recent runs
